# 🧩 QuizMast

**QuizMast** is a frictionless, AI-powered quiz platform designed for modern learners and developers[cite: 1, 2]. Built with a "Privacy-First" philosophy, it allows users to generate custom quizzes from simple prompts and test their knowledge without the friction of accounts or logins[cite: 17, 318, 319].

---

## ✨ Key Features

- **Prompt-to-Quiz Generation**: Instantly curate challenging questions on any topic using a simple AI prompt.
- **No-Login Experience**: Start learning immediately. All progress and history are stored locally in your browser .
- **Pro Testing Interface**: Features a 30-second timer per question, a visual progress bar, and interactive glassmorphism UI .
- **Detailed Analytics**: Get instant accuracy breakdowns with question-by-question review and color-coded feedback .
- **Modern Tech Stack**: Built with Next.js 15, Tailwind CSS, and Lucide React for a high-performance, sleek aesthetic .

---

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Glassmorphism & Dark Theme)
- **Icons**: [Lucide React](https://lucide.dev/)
- **AI Engine**: Advanced Prompt Engineering for dynamic quiz creation
- **Persistence**: Browser Local Storage (Privacy-Centric)

## 🚀 Getting Started

### Prerequisites

- Node.js 18.x or later
- A package manager npm

### Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/kritmanrao/QuizMast.git](https://github.com/kritmanrao/QuizMast.git)
   cd quizmast
   ```


2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Set up Environment Variables:**
   Create a `.env.local` file in the root directory and add your AI API keys :

   ```env
   AI_API_KEY=your_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result .
