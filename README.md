# 🧩 QuizMast

[cite_start]**QuizMast** is a frictionless, AI-powered quiz platform designed for modern learners and developers[cite: 1, 2]. [cite_start]Built with a "Privacy-First" philosophy, it allows users to generate custom quizzes from simple prompts and test their knowledge without the friction of accounts or logins[cite: 17, 318, 319].

---

## ✨ Key Features

- [cite_start]**Prompt-to-Quiz Generation**: Instantly curate challenging questions on any topic using a simple AI prompt[cite: 2, 22, 321].
- **No-Login Experience**: Start learning immediately. [cite_start]All progress and history are stored locally in your browser[cite: 17, 322, 323].
- [cite_start]**Pro Testing Interface**: Features a 30-second timer per question, a visual progress bar, and interactive glassmorphism UI[cite: 32, 38, 324].
- [cite_start]**Detailed Analytics**: Get instant accuracy breakdowns with question-by-question review and color-coded feedback[cite: 34, 49, 325].
- [cite_start]**Modern Tech Stack**: Built with Next.js 15, Tailwind CSS, and Lucide React for a high-performance, sleek aesthetic[cite: 215, 308, 326].

---

## 🛠️ Tech Stack

- [cite_start]**Framework**: [Next.js 15](https://nextjs.org/) (App Router) [cite: 20, 215, 327]
- [cite_start]**Styling**: [Tailwind CSS](https://tailwindcss.com/) (Glassmorphism & Dark Theme) [cite: 189, 203, 327]
- [cite_start]**Icons**: [Lucide React](https://lucide.dev/) [cite: 169, 327]
- [cite_start]**AI Engine**: Advanced Prompt Engineering for dynamic quiz creation [cite: 119, 327]
- [cite_start]**Persistence**: Browser Local Storage (Privacy-Centric) [cite: 122, 126, 327]

## 🚀 Getting Started

### Prerequisites

- Node.js 18.x or later
- A package manager npm

### Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/your-username/quizmast.git](https://github.com/your-username/quizmast.git)
   cd quizmast
   ```

````

2. **Install dependencies:**

   ```bash
   npm install

3. **Set up Environment Variables:**
   [cite_start]Create a `.env.local` file in the root directory and add your AI API keys[cite: 128]:

   ```env
   AI_API_KEY=your_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
   [cite_start]Open [http://localhost:3000](http://localhost:3000) with your browser to see the result[cite: 166, 328, 329].
````
